# 🌤️ Weather API Test Suite

A Postman-based test suite for practicing REST API testing, validation, and automation using a Weather API.

The project is organized into collections that test different weather endpoints while demonstrating API testing best practices, response validation, and workflow automation.

## Project Structure

```

.

├── .github/

│   └── workflows/

├── WeatherAPI Test Collection/

├── WeatherAPI-Collection/

└── README.md

```

## Features

- Test weather API endpoints

- Validate HTTP status codes

- Verify JSON response data

- Organize requests into Postman collections

- Support automated execution with GitHub Actions

- Practice REST API testing and validation

## Technologies

- Postman

- REST APIs

- JavaScript (Postman Test Scripts)

- GitHub Actions

- Git & GitHub

## Running the Collection

1. Clone the repository.

2. Import the Postman collection.

3. Configure any required environment variables.

4. Run the collection using the Postman Collection Runner or Newman.

## Skills Demonstrated

- REST API Testing

- API Response Validation

- Postman Collections

- Test Automation

- CI/CD with GitHub Actions

- Version Control with Git
